# QM640 Education Fee Burden Capstone

## Project Title
Predicting Education Fee Burden and Affordability Stress Among Indian Households Using NSS and CMS Education Microdata

## Overview
This repository contains the code, processed datasets, result tables, figures, and project documents for the QM640 Data Analytics Capstone. The study uses official Indian household survey microdata to estimate education fee burden and to predict whether a household falls into a high education fee burden category.

The project is based on:
- NSS 75th Round Household Social Consumption: Education
- CMS-E 2025 household education microdata
- An external all-India education expenditure time-series dataset

The main workflow is implemented in the notebook:
- `notebooks/Term3_Capstone.ipynb`

## Research Objective
The capstone aims to:
1. Construct a household-level education fee burden indicator.
2. Classify households into high-burden and non-high-burden groups.
3. Compare machine learning models for predicting high education fee burden.
4. Identify the main demographic, socioeconomic, and education-related drivers of fee burden.

## Folder Description

### `notebooks/`
Contains the main Jupyter notebook used for:
- loading data
- cleaning and merging datasets
- constructing fee burden measures
- exploratory data analysis
- model training and evaluation
- SHAP-based interpretation

### `data_raw/`
Contains raw NSS and CMS microdata files stored locally. These files are not committed to the public repository. See `data_raw/README.md` for details.

### `data_processed/`
Contains analysis-ready household-level datasets created from the raw microdata:
- NSS cleaned household dataset
- CMS cleaned household dataset
- cleaned national education expenditure trend dataset

### `results/tables/`
Contains exported model evaluation tables for NSS and CMS datasets.

### `results/figures/`
Contains saved figures used in the interim report and final analysis, including histograms, bar charts, ROC curves, and SHAP plots.

### `data_external/`
Contains external public datasets used for contextual analysis.

### `docs/`
Contains PDF versions of academic submission documents such as the synopsis and interim report.

## Data Sources
The project uses official NSS and CMS education-related microdata and a public education expenditure time-series dataset. Raw NSS/CMS data are not redistributed through this repository.

## Reproducibility
To reproduce the workflow:
1. Obtain the official raw NSS and CMS data files.
2. Place them inside the `data_raw/` folder using the filenames expected by the notebook.
3. Install the required Python packages.
4. Run `notebooks/Term3_Capstone.ipynb` from top to bottom.

## Requirements
Install dependencies using:

```bash
pip install -r requirements.txt
```
